Testing the BotForge Chat Widget

How to test:

1. Ensure your backend is running (`npm run dev` in `backend` folder).

2. Get your actual API Key from your BotForge Dashboard -> Embed tab.

3. Replace YOUR_API_KEY_HERE in this HTML file with your real key.

4. Open this test.html file in your browser.

5. The widget should appear in the bottom right corner!
